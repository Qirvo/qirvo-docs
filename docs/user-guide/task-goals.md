# Tasks & Goals

Comprehensive task management with goal tracking capabilities.

**Task Management:**

- Create, edit, and organize tasks
- Set priority levels (High, Medium, Low)
- Add due dates and deadlines
- Mark tasks as complete
- Organize tasks by categories or projects

**Goal Tracking:**

- Set long-term and short-term goals
- Break goals into actionable tasks
- Track progress over time
- Review and adjust goals regularly

**How to Use:**

1. Go to **Tasks & Goals** in the sidebar
2. Click "Add Task" to create new tasks
3. Set priorities and due dates
4. Use the goal section to define and track objectives
5. Regularly review and update your progress
