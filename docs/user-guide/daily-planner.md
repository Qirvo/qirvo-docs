# Daily Planner

The Daily Planner helps you structure your day with morning planning and evening reflection.

**Morning Planning:**

- Set daily priorities and focus areas
- Review scheduled tasks and appointments
- Plan your day's structure and goals

**Evening Review:**

- Reflect on the day's accomplishments
- Note lessons learned and areas for improvement
- Prepare insights for tomorrow's planning

**How to Use:**

1. Navigate to **Daily Planner** from the sidebar
2. Start with morning planning to set your intentions
3. Return in the evening for reflection and review
4. Use the prompts to guide your planning and reflection
