# Journal

Flexible journaling system inspired by the journal methodology.

**Features:**

- Rapid logging with bullets and symbols
- Monthly and weekly spreads
- Custom collections and trackers
- Migration of tasks and notes
- Search and filter entries

**How to Use:**

1. Navigate to **Journal**
2. Create daily, weekly, or monthly entries
3. Use bullet symbols for different entry types
4. Organize content with collections
5. Migrate incomplete tasks to future dates
