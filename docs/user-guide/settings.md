# Settings & Customization

Customize which features are available in your dashboard.

1. Navigate to **Settings**
2. Go to the "Modules" section
3. Enable/disable features based on your needs:
   - Tasks & Goals
   - Health Tracker
   - Bullet Journal
   - Projects
   - Echo CLI
   - AI Insights
   - Reminders
