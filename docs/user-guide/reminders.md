# Reminders

Schedule and manage notifications across all your activities.

**Features:**

- One-time and recurring reminders
- Multiple notification channels
- Integration with tasks and goals
- Flexible scheduling options
- Reminder history and management

**How to Use:**

1. Go to **Reminders** in the sidebar
2. Create new reminders with specific times
3. Set recurring schedules as needed
4. Choose notification preferences
5. Manage and edit existing reminders
