# Echo CLI

Command-line interface integration for power users.

**Features:**

- Terminal-style interface within the dashboard
- Quick task creation and management
- System commands and shortcuts
- Integration with other dashboard modules
- Customizable commands and aliases

**How to Use:**

1. Access **Echo CLI** from the navigation
2. Use text commands to interact with your data
3. Create tasks, notes, and reminders via CLI
4. Customize commands for your workflow
5. Access help documentation within the CLI
