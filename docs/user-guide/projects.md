# Projects

Track and manage your personal and professional projects.

**Features:**

- Project creation and organization
- Task breakdown and assignment
- Progress tracking and milestones
- File and resource management
- Collaboration tools (for team users)

**How to Use:**

1. Go to **Projects** in the sidebar
2. Create new projects with descriptions and goals
3. Break projects into manageable tasks
4. Set milestones and deadlines
5. Track progress and update status regularly

:::tip
For developers, you can connect your GitHub projects to each project and manage all your tasks from a single interface.
:::
